# Cli2Gui

A tool to convert command-line tools into GUI applications, built with Electron, Vue and TypeScript.

一个将命令行工具转换为图形界面应用的工具，使用 Electron、Vue 和 TypeScript 构建。

## Project Setup

### Install

```bash
$ pnpm install
```

### Development

```bash
$ pnpm dev
```

### Build

```bash
# For windows
$ pnpm build:win

# For macOS
$ pnpm build:mac

# For Linux
$ pnpm build:linux
```
